## Project Structure
I have organized the code into separate C# files to focus on specific concepts

### Basics & Control Flow
* `01_Hello_World.cs`: Setup, Main method, and basic output.
* `02_Variables_types.cs`: Data types (`int`, `double`, `decimal`), strict typing rules, and casting
* `03_Operators.cs`: Math operators, increment/decrement (`++`), and modulo logic
* `04_Control_Flow.cs`: User input, `if/else` logic, and `switch` statements
* `05_Loops.cs`: `for` loops, `while` loops
* `#1_LoopsExercices.cs`: Practice exercises for loop logic
* `06_ConditionnalOperator_&_Formatting.cs`: Ternary operators (`? :`) and string formatting string interpolation
* `07_Tryparse.cs`: Converting strings to numbers safely without crashing

### Data Structures & Strings
* `08_Strings.cs`: String manipulation methods (`Split`, `Substring`, `Replace`, etc)
* `#2_StringsExercices.cs`: Practice on text processing
* `09_Arrays.cs`: Creating and looping through fixed-size arrays
* `10_Lists&Dictionaries.cs`: Working with dynamic collections (`List<T>`) and Key-Value pairs (`Dictionary<T, T>`)
* `#3_ListsExercises.cs`: Exercises combining loops and collections

### Functions & Logic
* `11_Functions.cs`: Creating methods, return types, and scope
* `12_Parameters.cs`: Passing arguments, optional parameters, and named arguments
* `#4_Function&ParametersExercices.cs`: Practice exercises including `ref` and `out` parameters

### Advanced & OOP
* `13_ExeptionsHandling.cs`: Managing errors with `try`, `catch`, and `finally`
* `#5_ExeptionsHandling_Exercices.cs`: Creating custom `TryParse` logic using `out` parameters
* `14_Structures.cs`: Understanding `struct`
* `15_Classes.cs`: **Object-Oriented Programming**. Includes:
    * Fields vs Properties 
    * Constructors & `this` keyword
    * Lambda syntax (`=>`) for logic
    * Overriding `ToString()` and `Equals()`
